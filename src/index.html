About Me

Learning as I code

My interests with web designing started back in 2001 when I was in grade 6. There was a trendy website called AsianAvenue that allowed you to design your 'homepage' using pure HTML codes.

It was then when I found my passion in coding. I would always change how my 'page' looked and applied various styles. If I didn't know the code, I would right-click on another page, and click properties. This was before I knew about CSS and other languages. I have very fond memories with coding.

Fast-Forward to today, I finally finished my Front-End Web Developer course from SheCodes. This course made my interest grew in creating and designing websites. Here, you will find a showcase of my latest projects, skills, and interests in React.js, Responsive pages, as well as HTML and CSS. This page was built using bootstrap and mobile friendly. Feel free to explore and learn more about my work.
